Social Sharing Shortcode

[pmp-social-share] - use this shortcode to allow your website visitors to easily share your adoptable pets with their social networks.

Social Sharing Parameters

This shortcode does not accept parameters. Configuration is done thru the admin interface of your WordPress website using Tools/Monarch Settings.

NOTES: This shortcode requires a paid subscription.

Social Sharing Shortcode Best Practices

  1. Use this shortcode increase your animal's visibility by encouraging your website visitors to promote them through their social networks.
  2. Location - Enable the INLINE option.
  3. Networks - Add networks/options you believe are used by your website visitors.
  4. INLINE - Select the icon style and shape you would like to use. Select the option to Remove Icon Spacing.

Social Sharing Shortcode Example

[pmp-social-share]

This shortcode displays the social sharing options configured using the Monarch plugin.
