Lost & Found Search Shortcode

[pmp-lost-found-search] - Use this shortcode in a page of your website to display a combination of both lost AND found animal search results from PetPoint.

Lost & Found Search Parameters

Use of this shortcode requires the configuration of the option Lost/Found Combination Selection located on the Filter Options tab. This option specifies which method the shortcode will use when searching for and displaying results and is required when using this shortcode.

Lost & Found Search Shortcode Best Practices

  1. title - Include a call to action or other descriptive text to assist website visitors.
  2. species - Display all lost and found animals on a single page of your website and include speciesid as a filter option.
  3. orderby - Use the value "datelast" to display the most recent animals first. If not provided, results will be sorted by the field selected in the General Options tab.
  4. filter - Only include fields for which data is available.
  5. details - Include the minimum amount of information required to select an animal search result. Do not include name unless most lost/found animals are identifiable.

Lost & Found Search Shortcode Example

[pmp-lost-found-search title="Lost & Found Pets" orderby="datelast" filter="speciesid,sex,agegroup" details="sex,age,type,date"]

This shortcode displays search results for both lost AND found animals of all species in descending found date order with the title specified above the search results. Website visitors can filter by species, sex and age group. The animal sex, age, type and lost/found date are displayed for each search result.
