Animal Details Shortcode

[pmp-details] - include this shortcode in a page of your website to display the details of a specific animal selected from the AnimalsFirst search results or for use in creating a page for a specific animal.

Animal Details Parameters

Name
Values
Description
details - optional.
address, address_1, address_2, adopt_profile_url, adoption_fee, age, all_photos, altered, birthday, breed, city, country, est_birthday, declawed, description, display_age, featured, gender, google_address_url, google_geometry, google_formatted_address, google_lat, google_lng, id, intake_date, intake_type, kennel, location, long_address, los, microchip, name, outcome_date, primary_color, secondary_breed, secondary_color, seq_id, shelter_id, size, species, state, status, youtube_links, zip
Enclose the detail options for the animal details separated by commas in double quotes. The details are displayed in the order listed and override those configured in the Filter Options tab.
template - optional.
The name of the template file for use in displaying the animal details.
Paid feature. Enclose the name of the .php file in double quotes. Specifying a template file overrides the template configured in the General Options tab.

NOTES: The use of bold values requires a paid subscription. Review the AnimalsFirst documentation for a complete description for each field.

Animal Details Shortcode Best Practices

  1. details - All photos and the video will be displayed if available.
  2. template - This is an advanced feature requiring the coding of custom templates.

Animal Details Shortcode Example

[pmp-details details="name,species,gender,altered,breed,primary_color,birthday,declawed,description,size,seq_id,location"]

This shortcode displays details for a single animal with an identifier passed via the page URL displaying the listed fields in their specified order.
