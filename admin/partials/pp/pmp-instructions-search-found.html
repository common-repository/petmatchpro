Found Search Shortcode

[pmp-found-search] - Use this shortcode in a page of your website to display found animal search results from PetPoint.

Found Search Shortcode Best Practices

  1. title - Include a call to action or other descriptive text to assist website visitors.
  2. species - Display all found animals on a single page of your website and include speciesid as a filter option.
  3. orderby - Use the value "datelast" to display the most recent animals first. If not provided, results will be sorted by the field selected in the General Options tab.
  4. filter - Only include fields for which data is available.
  5. details - Include the minimum amount of information required to select an animal search result. Do not include name unless most found animals are identifiable.

Found Search Shortcode Example

[pmp-found-search title="Contact Us to Reclaim Your Found Animal" orderby="DateLast" filter="speciesid,sex,agegroup" details="founddate,sex,jurisdiction"]

This shortcode displays search results for found animals of all species in descending found date order with the title specified above the search results. Website visitors can filter by species, sex and age group. The animal founddate, sex and jurisdiction are displayed for each search result.
