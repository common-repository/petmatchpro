Featured Pet Shortcode

[pmp-adoptable-featured] - include this shortcode in a page of your website to highlight an animal configured within PetPoint as featured.

Featured Pet Parameters

Name
Values
Description
species - required.
all, barnyard, bird, cat, dog, horse, pig, rabbit, reptile, small&furry, other
Enclose a single value from the list in double quotes.
details - required - confirm.
adoptionapplicationurl, age, agegroup, altered, animalname, animaltype, arn, bannerurl, behaviorresult, behaviortestlist, bodyweight, buddyid, chipnumber, colorpattern, companyid, dateofbirth, declawed, dsc, featured, housetrained, id, lastintakedate, livedwithanimals, livedwithanimaltypes, livedwithchildren, location, memolist, nocats, nodogs, nokids, onhold, photo, prevenvironment, price, primarybreed, primarycolor, reasonforsurrender, secondarybreed, secondarycolor, sex, site, size, sn, specialneeds, species, stage, sublocation, timeinformerhome, videoid, wildlifeintakecause, wildlifeintakeinjury
Enclose the detail options for the animal details separated by commas in double quotes. The details are displayed in the order listed.
template - optional.
The name of the template file for use in displaying the animal details.
Paid feature. Enclose the name of the .php file in double quotes. Specifying a template file overrides the template configured in the General Options tab.
labels - optional.
no, yes.
Enclose a single value from the list in double quotes. The default is no if a value is not provided.

NOTES: The use of bold values requires a paid subscription. Review the PetPoint Web Services Guide for a complete description for each field.

Featured Pet Shortcode Best Practices

  1. details - The photo and videoid values are not required, they'll be displayed if available.
  2. template - This is an advanced feature requiring the coding of custom templates.
  3. If more than one animal of the specified species is configured, this shortcode will display the first one returned via the query to PetPoint.

Featured Pet Shortcode Example

[pmp-adoptable-featured species="dog" details="name,sex,age" labels="yes"]

This shortcode displays the name, sex and age with labels of the dog configured within Petpoint as featured.
