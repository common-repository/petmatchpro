Adoptable Search Shortcode

[pmp-adoptable-search] - Use this shortcode in a page of your website to display adoptable animal search results from PetPoint. The statuses included as "adoptable" are configured by your PetPoint administrator.

Adoptable Search Shortcode Best Practices

  1. species - Create a separate page on your website for each species of animal.
  2. title - Include a call to action or other descriptive text to assist website visitors.
  3. filter - Only include fields for which data is available. Do not include fields requiring visitors to enter text values.
  4. details - Include the minimum amount of information required to select an animal search result.

Adoptable Search Shortcode Example

[pmp-adoptable-search species="cat" title="Available Cats for Adoption" filter="sex,agegroup,nocats,nodogs,nokids,orderby" details="name,sex,age,location"]

This shortcode displays search results for adoptable cats with the title specified above the search results. Website visitors can filter by sex, agrgroup, nocats, nodogs and nokids and can sort the results by the options configured in the Filter Options tab. The animal name, sex, age and location are displayed for each search result.
