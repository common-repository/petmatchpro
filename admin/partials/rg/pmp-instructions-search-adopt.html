Adoptable Search Shortcode

[pmp-adoptable-search] - Use this shortcode in a page of your website to display adoptable animal search results from Rescue Groups.

Adoptable Search Shortcode Best Practices

  1. species - Create a separate page on your website for each species of animal.
  2. title - Include a call to action or other descriptive text to assist website visitors.
  3. filter - Only include fields for which data is available. Do not include fields requiring visitors to enter text values.
  4. details - Include the minimum amount of information required to select an animal search result.

Adoptable Search Shortcode Example

[pmp-adoptable-search animalspecies="cat" title="Available Cats for Adoption" filter="animalsex,animalgeneralage,animalokwithcats,animalokwithdogs,animalokwithkids,sortfield,sortorder" details="animalname,animalsex,animalgeneralage"]

This shortcode displays search results for adoptable cats with the title specified above the search results. Website visitors can filter by sex, general age, ok with cats, ok with dogs and ok with kids and can sort the results by the options configured in the Filter Options tab by the order selected. The animal name, sex and general age are displayed for each search result.
