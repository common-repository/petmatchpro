Adoptable Details Shortcode

[pmp-adoptable-details] - include this shortcode in a page of your website to display the details of a specific animal selected from the Rescue Groups search results or for use in creating a page for a specific animal.

Adoptable Details Parameters

Name
Values
Description
details - optional.
animalid, animaladoptionfee, animalaltered, animalbirthdate, animalbreed, animalcoatlength, animalcolor, animaldeclawed, animaldescriptionplain, animaleyecolor, animalfence , animalgeneralage, animalhousetrained, animalname, animalneedsfoster, animalokwithadults, animalokwithcats, animalokwithdogs, animalokwithkids , animalpictures, animalprimarybreed, animalsecondarybreed, animalsex, animalsizecurrent, animalsizeuom, animalspecialneeds, animalspecialneedsdescription, animalspecialdiet, animalspecies, animalstatus, animalsummary, animalthumbnailurl, animalurl, animalvideos, animalvideourls, animalyardrequired, locationname
Enclose the detail options for the animal details separated by commas in double quotes. The details are displayed in the order listed and override those configured in the Filter Options tab.
template - optional.
The name of the template file for use in displaying the animal details.
Paid feature. Enclose the name of the .php file in double quotes. Specifying a template file overrides the template configured in the General Options tab.

NOTES: The use of bold values requires a paid subscription. Review the Rescue Groups documentation for a complete description for each field.

Adoptable Details Shortcode Best Practices

  1. details - All photos and the video will be displayed if available.
  2. template - This is an advanced feature requiring the coding of custom templates.

Adoptable Details Shortcode Example

[pmp-adoptable-details details="animalname,animalspecies,animalsex,animalaltered,animalprimarybreed,animalcolor,animalbirthdate,animalhousetrained,animaldeclawed,animaldescriptionplain,animalsizecurrent,animalid,locationname"]

This shortcode displays details for a single adoptable animal with an identifier passed via the page URL displaying the listed fields in their specified order.
