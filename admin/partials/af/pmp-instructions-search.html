Animal Search Shortcode

[pmp-search] - Use this shortcode in a page of your website to display animal search results from AnimalsFirst.

Search Shortcode Best Practices

  1. type - Create a separate page on your website for each type of search.
  2. species - Create a separate page on your website for each species of animal.
  3. title - Include a call to action or other descriptive text to assist website visitors.
  4. filter - Only include fields for which data is available. Do not include fields requiring visitors to enter text values.
  5. details - Include the minimum amount of information required to select an animal search result.

Search Shortcode Example

[pmp-search type="available" species="cat" title="Available Cats for Adoption" filter="gender,age,sortfield,sortorder" details="name,gender,age"]

This shortcode displays search results for adoptable cats with the title specified above the search results. Website visitors can filter by gender and age and can sort the results by the options configured in the Filter Options tab by the order selected. The animal name, gender and age are displayed for each search result.
