Lost Details Shortcode

[pmp-lost-details] - include this shortcode in a page of your website to display the details of a lost animal selected from the PetPoint search results or for use in creating a page for a specific lost animal.

Lost Details Parameters

Name
Values
Description
details - optional.
age, animalname, animaltype, bodyweight, bodyweightunit, city, coat, collarcolor, collarcolor2, collartype, collartype2, colorpattern, companyid, declawed, dsc, ears, eyes, id, jurisdiction, lostdate, lostlocation, marks, primarybreed, primarycolor, secondarybreed, secondarycolor, sex, size, species, state, tail
Enclose the detail options for the animal details separated by commas in double quotes. The details are displayed in the order listed and override those configured in the Filter Options tab.
template - optional.
The name of the template file for use in displaying the animal details.
Paid feature. Enclose the name of the .php file in double quotes. Specifying a template file overrides the template configured in the General Options tab.

NOTES: The use of bold values requires a paid subscription. Review the PetPoint Web Services Guide for a complete description for each field.

Lost Details Shortcode Best Practices

  1. details - All photos will be displayed if available.
  2. template - This is an advanced feature requiring the coding of custom templates.

Lost Details Shortcode Example

[pmp-lost-details details="species,primarybreed,sex,age,lostdate,jurisdiction,primarycolor,bodyweight,declawed,id"]

This shortcode displays details for a single lost animal with an identifier passed via the page URL displaying the listed fields in their specified order.
