Integrates animal search and details from your PetPoint/Petango, RescueGroups or AnimalsFirst account into your website with simple shortcodes.

== Description ==

PetMatchPro now includes support for RescueGroups adoptable animals as well as adoptable, found and lost animals sourced from PetPoint/Petango and AnimalsFirst.

PetMatchPro provides your website visitors with an integrated experience in viewing, selecting, and ultimately adopting one of your animals configured in the data management system you use to run your shelter or rescue.  

But it doesn’t stop there!

At the same time, it collects and shares the data you need to improve your ability to find forever homes for more animals.

+ PetMatchPro is a comprehensive visibility, tracking and reporting solution for your shelter or rescue website.
+ PetMatchPro is a simple-to-use solution to brand the animal search and display on your website.
+ PetMatchPro is an affordable solution with a low monthly investment (for the Junior and Preferred Plans).
+ PetMatchPro is an evolving offering that will add new functions and features that you require.
+ PetMatchPro is something you can try for free by registering for the Basic Plan.

== Installation ==

1. Upload the current petmatchpro plugin zip file to your `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. View the PetMatchPro Instructions tab for additional steps to configure and use the plugin.
