PetMatchPro Plugin Download

  1. Login to your PetMatchPro account.
  2. Select the Downloads tab.
  3. Click the link for the highest version of the petmtachpro-vx.y.z.zip file displayed in the listing.

Social Sharing Plugin Download

  1. Login to your PetMatchPro account.
  2. Select the Downloads tab.
  3. To download the plugin, click the link for the file petmatchpro-social-share-plugin.zip displayed in the listing.
  4. To download the plugin settings file, click the link for the file petmatchpro-social-share-plugin-settings.zip displayed in the listing.

PetMatchPro Plugin Activation

  1. Login to your website as an administrator.
  2. Follow the steps required to add a new plugin from a zip file.
  3. Once the file has been uploaded, activate the PetMatchPro plugin.
  4. Navigate to PetMatchPro/License Information in the side menu of your website administration area.
  5. Enter the License Code received via email after registering for the plugin and your registration email and click the Activate button.

Social Share Plugin Activation

  1. Extract the settings file from the zip file petmatchpro-social-share-plugin-settings.zip.
  2. Login to your website as an administrator.
  3. Follow the steps required to add a new plugin from a zip file.
  4. Once the file petmatchpro-social-share-plugin.zip has been uploaded, activate the Monarch plugin.
  5. Navigate to Tools/Monarch Settings in the side menu of your website administration area.
  6. Select the import/export icon (up and down arrow) from the heading of the Monarch settings.
  7. Click the Choose File button within the import section and navigate to the settings file you extracted in the first step.
  8. After the file is selected the Import button will be enabled. Click the Import button to configure the Monarch Social Sharing plugin.
  9. NOTE: USE OF THE SOCIAL SHARE FEATURE REQUIRES A PAID SUBSCRIPTION.
