Animal Detail Shortcode

[pmp-detail] - Use this shortcode to display a single detail of a specific animal from AnimalsFirst.

Animal Detail Parameters

Name
Values
Description
detail - required.
address, address_1, address_2, adopt_profile_url, adoption_fee, age, all_photos, altered, birthday, breed, city, country, est_birthday, declawed, description, display_age, featured, gender, google_address_url, google_geometry, google_formatted_address, google_lat, google_lng, id, intake_date, intake_type, kennel, location, long_address, los, microchip, name, outcome_date, primary_color, secondary_breed, secondary_color, seq_id, shelter_id, size, species, state, status, youtube_links, zip
Enclose the desired detail in double quotes.
case - optional.
The format to display the animal detail.
Enclose the value lower, mixed or upper in double quotes.
replace - optional.
Character(s) used to replace spaces in the animal detail.
Enclose the character(s) in double quotes.

NOTES: The use of bold values requires a paid subscription. Review the AnimalsFirst documentation for a complete description for each field.

Animal Detail Shortcode Best Practices

  1. Use this shortcode to customize the heading and/or copy of your website page displaying animal details for any search method (adoption, lost or found).
  2. detail - Only a single detail can be used as a parameter.
  3. case - The detail is displayed as lower case if the case parameter is not included.
  4. replace - Used only when the data is poorly formatted.

Animal Detail Shortcode Example

[pmp-detail detail="name" case="mixed"]

This shortcode displays an animal's name in mixed case (first letter capitalization).
